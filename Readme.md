# Walmart Sales Dashboard

This Excel file contains a Walmart Sales Dashboard with the following sheets:

1. **Actual Data** – Raw sales data.
2. **Working Data** – Cleaned and processed data for analysis.
3. **Pivot Tables** – Pivot tables used in the dashboard.
4. **Dashboard** – Visuals including charts and KPIs.
5. **KPI** – Key Performance Indicators summary.

**Instructions:**
- Open the Excel file in Microsoft Excel (2016 or later recommended).
- Navigate to the 'Dashboard' sheet to view charts and KPIs.
- Pivot tables are in the 'Pivot Tables' sheet for further analysis.
